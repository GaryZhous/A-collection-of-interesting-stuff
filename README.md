# A Collection of Interesting Stuff

A personal reference library of algorithms, data structures, system design implementations, and language fundamental.

---

## Contents

### 🟦 Go

| File | Description |
|------|-------------|
| `BankSys.go` | Bank transaction system supporting transfers, deposits, and withdrawals with balance validation |
| `DupeFolder.go` | Duplicate folder detection and deletion using a Trie-based approach |
| `Interface.go` | Go interface implementation and polymorphism with multiple method signatures |
| `LinkedList.go` | Classic singly-linked list with append and traversal |
| `SearchSys.go` | Product search autocomplete using binary search and prefix matching |
| `SimpleAPI.go` | HTTP API using chi router with Stripe payment integration |
| `Sudoku.go` | Sudoku solver using backtracking with validity checking |
| `datastructure.go` | Generic stack and heap implementations |
| `generics.go` | Generic utility functions for summing slices and finding max elements |
| `goreview.go` | Go fundamentals: variables, types, functions, and control flow |
| `goroutine.go` | Concurrency with goroutines, WaitGroups, and synchronization patterns |
| `minCut.go` | Minimum palindrome cuts via dynamic programming |
| `oop.go` | OOP concepts using structs, methods, and interface-based polymorphism |
| `osexample.go` | OS interactions: environment variables, directories, and file operations |
| `tricks.go` | Bit manipulation: shifts, XOR swaps, power-of-2 operations |
| `wildcard.go` | Wildcard pattern matching (`?` and `*`) using dynamic programming |

## Topics Covered

- **Data Structures**: Linked list, stack, deque, trie, skip list, union-find, heap
- **Algorithms**: BFS, DFS, backtracking, dynamic programming, bit manipulation, prefix sums
- **Caching**: LRU, LFU, lazy cache
- **System Design**: File systems, bank systems, ride-sharing, auctions, transit systems, search autocomplete
- **Concurrency**: Goroutines, Rust async/threads, Java executors, locks and semaphores
- **Language Internals**: Generics, OOP patterns, smart pointers, OS interactions
